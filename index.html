LaTeX source, R scripts, JAGS scripts, and associated files for a short, introductory workshop on Bayesian inference using R and JAGS. Presented by Kent Holsinger and Xiaojing Wang, University of Connecticut as part of Plant Diversity in the GCFR: From Genomes to Biomes, 20-22 July 2015, Colophon Room, SANBI, Kirstenbosch, South Africa.

Links to software we will use during the workshop (available for Windoze, Mac OS X, Linux, and other platforms):

  • Required (if you want to follow along during the workshop)
    • R http://cran.r-project.org/
    • JAGS http://sourceforge.net/projects/mcmc-jags/
    • R2jags package within R
      install.packages("R2jags")

  • Optional (interfaces for working with R)
    • RStudio http://www.rstudio.com/ - provides a simple, easy to use environment to work with R
    • ESS http://ess.r-project.org/ - Emacs Speaks Statistics. Emacs is an extremely powerful text editor, and it is available for many platforms. ESS is a mode within Emacs that provides many of the features of RStudio.

The data file is from Mitchell et al., The American Naturalist, 185:525-537; 2015 http://dx.doi.org/10.1086/680051 and Mitchell et al. http://dx.doi.org/10.5061/dryad.sc286.
